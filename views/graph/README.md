# Graph Views

Use Mermaid exports for shareable diagrams.

In Obsidian:

- Run the P0 pipeline first with `import-paper`, `compile-paper`, or `compile-batch`.
- Open `views/canvas/starter.canvas` to see a ready-made visual canvas.
- Open `views/dataview/compilation-queue.md` to inspect which raw papers are still waiting for compilation.
- Click the left sidebar graph icon to open the global graph view.
- Use the command palette and run `Open local graph` when you want a graph centered on the current note.
- If note properties take over the page, collapse the properties panel or scroll below it to reach the note body and Mermaid blocks.
